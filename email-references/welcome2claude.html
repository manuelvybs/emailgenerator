Your Claude Code journey starts now
Welcome to Claude Code, your AI coding collaborator. Let's get you to your first commit in the next 5 minutes.
Step 1: Let Claude Code learn your project
Open your project directory and run `claude`.

Then, type:

> /init

Claude will create a CLAUDE.md file that captures how your project works—build commands, testing setup, best practices. 

Then, commit:

> Use github CLI to commit the CLAUDE.md file
Step 2: Make your first change
Ask Claude to improve something small, like making a targeted code change, fixing a bug, or adding some tests.

> Add a button for users to give feedback. Make sure to add tests and check that they pass.

Give Claude Code feedback as it works, or hit `shift + tab` to auto-accept future edits.
Step 3: Commit it
Once your code changes are done, tell Claude Code:

> Commit and push this change
